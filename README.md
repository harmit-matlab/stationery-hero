# PenCraft Stationery

A stationery-themed web application built with a modern Nx monorepo stack.

## Tech Stack

- **NestJS** – Backend API framework
- **Next.js** – Frontend React framework
- **Prisma** – Database ORM
- **PostgreSQL** – Relational database
- **Tailwind CSS** – Utility-first styling

## Getting Started

### Prerequisites

- Node.js (see `.nvmrc` for version)
- pnpm

### Install

```bash
pnpm install
```

### Run

```bash
# Start the full stack
pnpm serve

# Start the frontend only
pnpm serve-frontend
```

## Features

- Warm, craft-inspired stationery theme with OKLCH color palette
- Internationalization (i18n) support
- Reusable UI component library
- Responsive layout with hero section
